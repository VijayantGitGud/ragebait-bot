# Problem Definition

This project aims to identify messages in group chat environments that exhibit linguistic and contextual patterns commonly associated with rage-baiting.

Rage-baiting is defined here as opinionated language framed in a way that is likely to provoke emotional or reactive responses rather than encourage constructive discussion.

The system does not attempt to infer the author’s intent, emotional state, or factual correctness of the message. Instead, it outputs a probabilistic score indicating the likelihood that a message contains provocative framing based on learned linguistic signals.

The system is designed to provide soft, humorous feedback intended to encourage reflection and de-escalation, rather than moderation, punishment, or message removal.
