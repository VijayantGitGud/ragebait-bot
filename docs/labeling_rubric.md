# Rage-Bait Annotation Rubric

This rubric defines how human annotators assign rage-bait scores to messages.

The score represents the degree to which a message uses provocative framing likely to elicit emotional or reactive responses.

## Scoring Scale

| Score | Description                                                |
| ----- | ---------------------------------------------------------- |
| 0.0   | Neutral, factual, or purely informational                  |
| 0.3   | Opinionated but framed constructively                      |
| 0.6   | Provocative phrasing that may encourage reactive responses |
| 0.9   | Strong rage-bait characteristics                           |

## Common Linguistic Indicators

Annotators should look for combinations of the following signals:

- Absolutist language (e.g., "everyone knows", "always", "never")
- Broad generalizations about groups or identities
- Provocative challenges (e.g., "prove me wrong", "cope", "cry about it")
- Emotionally loaded adjectives without supporting evidence
- Dismissive or mocking tone

## Annotation Notes

- Annotators should focus on language and framing, not personal agreement or disagreement.
- Intent should not be assumed.
- If uncertain, choose the lower score and add a note explaining ambiguity.
